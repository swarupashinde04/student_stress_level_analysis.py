# Student Stress & Burnout Risk Analyzer

**Author:** Swarupa Shinde  
**Level:** Undergraduate (1st Year, B.Tech)  
**Domain:** Student Well-being | Decision Support Systems

## 📌 Problem Statement
Academic pressure, lack of sleep, and frequent evaluations can lead to high stress and burnout among students. However, stress is often ignored until it affects performance and health.

## 💡 Proposed Solution
This project presents a simple rule-based system to estimate a student's stress level using basic academic and lifestyle inputs.

## ⚙️ Methodology
The system takes the following inputs:
- Daily study hours
- Daily sleep hours
- Number of assignments per week
- Days left for the next exam

A stress score is computed using predefined logical conditions and classified into:
- LOW STRESS
- MODERATE STRESS
- HIGH STRESS

Based on the classification, practical recommendations are generated.

## 🧠 Core Logic
- Higher study load and assignment frequency increase stress
- Reduced sleep significantly increases stress
- Exam proximity adds psychological pressure
- Total stress score determines risk category

## 📊 Output
The system provides:
- Stress score
- Stress level classification
- Action-oriented recommendation

## 🛠️ Technology Used
- Python
- Conditional logic
- Console-based input/output

## ⚠️ Limitations
- Rule-based (no real psychological data)
- Does not account for individual stress tolerance
- Uses fixed thresholds
